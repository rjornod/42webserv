# Webserv

*This project has been created as part of the 42 curriculum by rojornod and dloustal*

## Description

A simple HTTP web server written in C++

The goal of this project is to understand how web servers work by building one from scratch. It focuses on networking, HTTP communication, socket programming and server architecture.

The repository is currently a work in progress. Further updates will be published as the project evolves.

## Instructions

### Build

```bash
make
```

### Run

